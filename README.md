### Lonely Planet

## Before you run it
* ``` install ruby 2.3.0 ```
* ``` bundle install ```

## Running
# Basic Info
* The app takes 2 files as an input -```destinations.xml``` and ```taxonomy.xml```.
* To start type: ```ruby bin/program.rb -t TAXONOMY_FILE_PATH -d DESTINATION_FILE_PATH```
* The app saves an output in ```/output``` directory.

## Testing
* ``` rspec spec/ ```
